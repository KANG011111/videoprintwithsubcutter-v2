🎯 標題隱藏功能測試步驟

步驟 1：基本功能測試

  1. 在主程式中選擇「模式 3 - 文章卡片樣式」
  2. 在「🏷️ 卡片標題」欄位輸入測試標題，如：「AI 提示技巧」
  3. 上傳測試檔案並生成截圖
  4. 觀察卡片是否正確顯示標題

步驟 2：標題隱藏測試 ⭐

  1. 點擊任一卡片的標題區域開始編輯
  2. 完全清空標題內容（刪除所有文字）
  3. 重點檢查：標題區域和橫線是否完全消失
  4. 確認圖片區域間距是否正確調整

步驟 3：重新顯示測試

  1. 在空的標題欄位重新輸入文字
  2. 確認標題區域和橫線重新出現
  3. 檢查佈局是否恢復正常

🔍 除錯資訊

如果標題無法正確隱藏，請：

  1. 按 F12 開啟開發者工具
  2. 查看 Console 標籤是否有「標題變更檢測」訊息
  3. 檢查 Elements 標籤中的 .title-section 元素樣式
  4. 確認是否有 CSS 類別 .hidden 或 .visible
